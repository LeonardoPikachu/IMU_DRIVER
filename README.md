# IMU_DRIVER
imu AH100B in ROS

## 1. Prerequisites
1.1 **Ubuntu** and **ROS**  

1.2 ROS package: **serial** is a must. Using the following cmd to install if needed:  
  ```sudo apt-get install ros-<distro>-serial```
    
## 2. Additional
  Beware of a self-defined ROS package in the same name 'serial'.
